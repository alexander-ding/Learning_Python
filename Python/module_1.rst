模组1：Python入门
=====================================
本模组将向读者介绍Python以及一些强大的Python编辑以及运行工具。我们将讨论Python是什么，编程语言具体是什么意思，以及安装Python具体对你的电脑意味着什么。

其次，我们将一步步介绍如何安装Python。具体来讲，我们会安装Anaconda版本的Python。其中含有很多对科学工作极其重要的第三方模组和工具（比如NumPy，matplotlib，Jupyter）。

不管你准备好了没有，我们该写一些代码了！当我们安装好了全新的Python之后，我们将进入一个比较非正式的Python教程--随便试试数字，写几个字符串，玩玩列表。这将帮助我们建立一个对Python语法的直观影响以及体验写和运行Python代码的感觉。模组2则是一个更加细致系统的对Python的学习。

最后，我们会花一些时间去勘察一些常用的写Python代码的工具。这包括Jupyter notebook，一个在实验者，数据科学家，教育家，和学生之间非常流行的工具。同事，我们也会介绍两个非常强大的Python集成开发环境（IDE）：PyCharm和Visual Studio Code。
